## 📝 使用说明

### 平台支持标识
- 🏠 - 本地运行
- ☁️ - 云服务
- 🍎 - macOS
- 🪟 - Windows  
- 🐧 - Linux
- 🎖️ - 官方实现

### 语言标识
- 🐍 - Python
- ☕ - Java
- 🦀 - Rust
- 🦎 - Go
- 📇 - TypeScript/JavaScript

### 安装和使用
1. 选择适合您需求的MCP服务器
2. 按照GitHub仓库的README说明进行安装
3. 配置MCP客户端（如Claude Desktop）
4. 开始使用MCP工具
